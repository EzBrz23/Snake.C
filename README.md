#  Snake Game em C

Jogo da cobrinha feito em C usando console do Windows.

## Features
- Sistema de pontuação
- Ranking salvo em arquivo
- Frutas com efeitos:
  -  Maçã (+2)
  -  Uva (+5)
  -  Banana (+10 e velocidade x2 por 3s)
  -  Bônus (+ 100 e imunidade por 5s)
- Painel lateral (HUD)
- Menu interativo

## Tecnologias
- C
- Windows API (console)

##  Como rodar
Compile com:
gcc main.c -o snake

Execute:
./snake
